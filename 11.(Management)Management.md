# Managemant

## Monitoring

- cloud의 모든 리소스를 모니터링
- 기본,상세 모니터링으로 나뉜다.

## Sub Account

- root 계정에서 계정별 권한을 부여하여 리소스 관리에 도움을 주는 서비스
- 2차인증 제공
- 별도의 페이지를 통해 로그인 진행

## WMS

- 시나리오 기반으로 웹 서비스를 테스트 해주는 서비스
- 오류 감지시 알림 제공
- 성능등급,측정위치,로드시간,페이지크기,요청건수,에러이벤트 등을 측정해준다

## Cloud Activity Tracer

- 다양한 계정 활동 로그 수집
- CLA와 연동되어 사용된다.

## Resource Manager

- Cloud 내 리소스를 통합 관리하는 서비스
